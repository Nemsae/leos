# lEOS

## SETUP
1. `$ git clone https://github.com/Nemsae/leos`
2. `$ cd leos`
3. `$ npm i`
4. `$ cd client && npm i`
5. `$ cd..`
6. `$ npm start`

### Encrypted messages are persisted on a MongoDB server.
